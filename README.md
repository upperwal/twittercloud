### Tweet Cloud

#### Requirement

* Bootstrap 3.*
* JQuery 1.*

Include .js and .css files into your project and initiate Twitter Cloud using TwitterCloud('#id or .class') in your js file. 

#### For Twitter Wall

Use TwitterCloud('#id or .class') or TwitterCloud('#id or .class', type='wall')
